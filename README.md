# API Livraria

Uma API CRUD para uma livraria, projeto em andamento...

## Stack utilizada

- Node
- Express
- Mongoose

## Aprendizados

- NodeJS
- Express
- MongoDB

## Rodando localmente

- Clone o projeto

- Entre no diretório do projeto

- Instale as dependências

```bash
  npm install
```
- Configure o dbConnect para o seu banco de dados especifico

- Inicie o servidor

```bash
  npm run dev
```
