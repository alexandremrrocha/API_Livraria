# API Livraria

Uma API CRUD para uma livraria, co

## Stack utilizada

- Node
- Express
- Mongoose

## Aprendizados

- NodeJS
- Express
- MongoDB

## Rodando localmente

- Clone o projeto

- Entre no diretório do projeto

- Instale as dependências

```bash
  npm install
```

- Configure um banco de dados de sua preferencia

- Inicie o servidor

```bash
  npm run dev
```
